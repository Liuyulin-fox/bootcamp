環境変数とは、
・プログラムの挙動を調整するためのパラメータの一種
・プロセス毎のパラメータ、マルチスレッドプログラム間の共有データ
  ※マルチスレッドプログラム：アプリケーションのプロセス（タスク）を複数のスレッドに分けて並行処理する方式

環境変数の種類はシェル変数、環境変数があり、各々特徴がある
・シェル変数：設定または定義されたシェル内にのみ含まれている変数
  ※子プロセスには渡されるものではない
・シェル変数のリストを表示する方法は set コマンドを使用 
  ex）set | grep TEST_VAR
・シェル変数を環境変数に変換するコマンドは export を活用 
  ex）export TEST_VAR

・環境変数：子シェルまたはプロセスによって引き継がれる変数
・環境変数のリストを表示する方法は env または printenv コマンドを使用 
  ex）printenv | grep TEST_VAR
・環境変数を格下げ「シェル変数に戻す」と設定方法は export -n 変数名にシェル変数に変換できる
  ex）export -n TEST_VAR
